
The glfw target is for building native C++ GL apps.

This uses the following APIs:

GLFW - A nice GLUT-like OpenGL wrapper: http://www.glfw.org/

OPENGL - for rendering: http://www.opengl.org

OPENAL - for audio: http://www.openal.org

STB_IMAGE - Sean Barrett's very cool image loading routines: http://nothings.org

Requirements:

For Windows, you will need Visual C++ 2010 Express

For Macos, you will need XCode 3.2.5
